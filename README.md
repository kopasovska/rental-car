# RentalCar Project

A car rental platform built with Next.js that allows users to browse, search, and rent cars easily.

## Live Demo

Check out the project live: [RentalCar Live Demo](https://rental-car-tau-inky.vercel.app/)

## Usage

- Browse cars
- Search for available cars
- Click on a car to see details
- Rent a car online

## Features

- Responsive design
- Next.js 13 App Router
- Dynamic routing for car details
- Open Graph metadata for social sharing
- Link prefetching for faster navigation

## Tech Stack

- Next.js
- React
- TypeScript
- CSS Modules

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
